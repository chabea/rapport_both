Description du mode de fonctionnement

L'élaboration de notre projet s'est reposé sur plusieurs point pour l'atteinte des résultats énuméré dans le cahier de charge. Dans la section avant celle-ci nous avons listé les membres du groupe ranger par section (Backend, Frontend,...).

Ce découpage nous a beaucoup aidé dans l'organisation, car elle a permis à chaque membre du groupe de s'investir en se concentrant sur la tâche confié et de bien le faire.

Dans la section qui suis, nous allons présenté ce à quoi consiste chaque découpage lié à notre mode de fonctionnement.

Backend

C'est la partie qui s'occupe du traitement des données de notre application. Nous avons utilisé python avec le framework DJANGO pour la structure et la logique de notre code Backend. Le découpage s'est fais en fonction du niveau de familiarité avec la technologie.

Frontend

C'est la partie visible de notre application. Nous avons utilisé les notions de bases: HTML, CSS, Javascript et le framework Bootstrap pour la mise en forme de notre application. L'équipe s'est réparti les tâches liés aux différentes interface pour le gérer dans le temps imparti.

Identité visuel

Cette partie a consisté à faire appel à la créativité des membres du groupe. Nous avons à ce niveau, conçu un logo, faire le choix des couleurs, le choix de la police. Il nous a permis de rentrer dans la peau du visitteur et de produire un environnement qui lui sera chalereux.

Conception

Ici nous avons eu à faire appel à toute la logique qu'il faut pour énumérer les acteurs de notre système. Nous avions aussi ressortir des analyses les différentes tables à créer pour notre base de données. Cette étape était très essentiel pour une cohérence dans l'esprit de tous. Et facilite le développement Backend.

Rapport

Nous avons fais une synthèse sur l'objectif général du projet et ses objectifs précis. Cela a consisté à décrire la méthodologie utilisée, les étapes de conception, les étapes de développement, les étapes liées à l'identité visuel de notre plateforme ainsi que les étapes de déploiement du projet.

Manuel d'utilisation

Nous avons essentiellement eu à présenter nottre application et la manière de l'utiliser. Les fonctionnalités principale ont été présenter dans le manuel d'utilisation, ainsi que le mode de déploiement.
